# Webアプリ作成指示（共通）

以下の作業をすべて実行してください。
この指示内の <APP_DIR> は「対象アプリ名」に置き換えてください。
1) Viteのvanilla-tsテンプレートで <APP_DIR> を作成
2) 依存関係をインストール
3) <APP_DIR>/index.html, <APP_DIR>/src/style.css, <APP_DIR>/src/main.ts を実装
4) 技術スタックは TypeScript + HTML + CSS（Vanilla、フレームワーク不使用）
5) アーキテクチャは Vite の標準構成（HTML=構造、CSS=見た目、TS=ロジック）
6) 状態管理は `src/main.ts` 内の単一オブジェクトで保持する
7) カレンダーから日付を選択し、テキスト欄にTodoを入力してボタンで登録できるようにする
8) 登録したTodoを日付の昇順かつ登録順で一覧表示する
9) チェックボックスで未完／完了を管理できるようにする
10) 完了したTodoをボタンで一括削除できるようにする
11) ローカル開発サーバを起動
12) テストを実行し、起動手順と期待結果を箇条書きで示す
13) 動作確認は Vite が出力するURLを使う
